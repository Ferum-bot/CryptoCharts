# CryptoCharts

#### Android application that can show cryptocurrency charts in real time.

#### Using open WebSocket API from [Coinbase](https://docs.pro.coinbase.com/#websocket-feed).

### Using libraries:
* AndroidX
* Java WebSockets
* Adapter Delegates
* Moshi

### Using features:
* MVVM
* View Binding
* RecyclerView with DiffUtil
* WebSockets
* Custom Views
